Files included in this directory include a CMakeLists.txt that tells 
ros things that needed to be included during the compilation

Our report HW3_Report.pdf

The package.xml imports and runs the correct ros packages. 

And there is a src directory that contains the .cpp file that
contains all of our source code.

In order to run the code you need to have a working catkin_ws on
a raspberry pi/turtlebotthen you move the project 4 folder onto 
the raspberry pi.  Then you catkin_make the directory and run
rosrun project4 project4 on the terminal.
